⚡ Step 1 — Cloudflare Worker Setup (one-time)

This app fetches real Yahoo Finance data through a tiny Cloudflare Worker you host for free. Takes about 2 minutes.

  1. Go to workers.cloudflare.com and sign up (free)
  2. Click Create a Worker
  3. Delete all the default code and paste the contents of worker.js (included in the download)
  4. Click Deploy
  5. Copy your Worker URL — it looks like your-worker.username.workers.dev
  6. Paste it below and click Save
